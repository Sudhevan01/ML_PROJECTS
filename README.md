🚀 Data Science Major Projects

A collection of 7 Data Science & Machine Learning projects built using Python and popular ML libraries.
These projects focus on data analysis, visualization, machine learning, NLP, clustering, recommendation systems, and forecasting.

📌 Projects Included
1️⃣ Titanic Survival Prediction

Predict passenger survival using classification algorithms.

2️⃣ Iris Flower Classification

Classify Iris flower species using ML models.

3️⃣ House Price Prediction

Predict house prices using regression techniques.

4️⃣ Customer Segmentation

Group customers using clustering algorithms like K-Means.

5️⃣ Movie Recommendation System

Recommend movies based on user preferences.

6️⃣ Sentiment Analysis of Tweets

Analyze tweet sentiments using NLP techniques.

7️⃣ Sales Forecasting

Forecast future sales using time series analysis.
